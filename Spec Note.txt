# Warehouse Management App
This is a project playground for everyone collaborating. The application will manage the movement of Stock in-out and within several warehouses.

## Getting Started
Olympus is a project of microservices. Hera is the client and Zeus is the server. This application should be deployable to any platform.

### Prerequisites 
Dear collaborators, please update the version & technologies which related to your works.
All the specs below should be in minimum version.

### Specs
Language: Java JDK 1.9, C++ 11

Server: Spring Boot 2.0 or Spring Framework 5, embedded Tomcat 8

Build tool: Maven, Gradle

Mobile: Xamarin vXX

Desktop: Qt5

Message broker: Active MQ

Database: Postgres vXX

ORM: JPA & Hibernate 5
